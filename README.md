# 🚀 Desafio GitHub Starter - Ganhe seus Adesivos!
[Códigos GIT](https://gist.github.com/leocomelli/2545add34e4fec21ec16)
Seja bem-vindo(a)! Se você chegou até aqui, está a poucos passos de aprender a colaborar no GitHub e garantir os adesivos para o seu notebook. 🎉

---

## 📋 Como Participar?

1. Certifique-se de estar logado na sua conta do GitHub.
2. Clique no ícone de **lápis** (Editar este arquivo) no canto superior direito.
3. Vá até a tabela abaixo e adicione uma nova linha com o seu **Nome Completo**.
4. Role a página até o final, clique em **Commit changes...** (ou "Propose changes") para salvar.
5. Pronto! Agora é só falar com a organização para pegar seus adesivos. 🎫
